📋 สรุปเทคนิคการหาลิมิต
1. แทนค่าตรงๆ ได้เลย (ฟังก์ชันต่อเนื่อง)
ถ้า $f$ ต่อเนื่องที่ $a$: $\lim_{x\to a} f(x) = f(a)$
ตัวอย่าง: $\lim_{x\to 3}(2x+1) = 7$
2. รูป 0/0 → แยกตัวประกอบ
ถ้าแทน $x=a$ แล้วได้ $0/0$ ลองจัดรูปให้ $(x-a)$ หักล้างกัน
$$\lim_{x\to 2}\frac{x^2-4}{x-2} = \lim_{x\to 2}\frac{(x-2)(x+2)}{x-2} = 4$$
3. รูป 0/0 ที่มีราก → คูณคอนจูเกต
$$\lim_{x\to 0}\frac{\sqrt{x+4}-2}{x} = \lim_{x\to 0}\frac{1}{\sqrt{x+4}+2} = \frac{1}{4}$$
4. ลิมิตพื้นฐานที่ต้องจำ
$$\lim_{x\to 0}\frac{\sin x}{x} = 1, \quad \lim_{x\to 0}\frac{1-\cos x}{x} = 0$$ $$\lim_{x\to 0}\frac{e^x - 1}{x} = 1, \quad \lim_{x\to 0}\frac{\ln(1+x)}{x} = 1$$ $$\lim_{n\to\infty}\left(1+\tfrac{1}{n}\right)^n = e$$
5. รูป ∞/∞ → L'Hospital (ขั้นสูง)
ถ้า $\lim \dfrac{f}{g}$ เป็นรูป $\dfrac{0}{0}$ หรือ $\dfrac{\infty}{\infty}$: $$\lim\frac{f(x)}{g(x)} = \lim\frac{f'(x)}{g'(x)}$$
6. เข้าใกล้ ∞ → พิจารณาเทอมโตที่สุด
$$\lim_{x\to\infty}\frac{3x^2+1}{5x^2-x} = \frac{3}{5}$$

∫ ปริพันธ์ (Integration)

จากลิมิตของผลรวมรีมันน์ → แอนติเดริเวทีฟ → ถึงทฤษฎีบทหลักของแคลคูลัส (FTC)

📊 แนวคิด: แบ่งพื้นที่เป็นสี่เหลี่ยมเล็กๆ

พื้นที่ใต้กราฟของ $f$ บน $[a,b]$ หาได้โดยแบ่งช่วงเป็น $n$ ส่วน และบวกพื้นที่ของสี่เหลี่ยมแต่ละแท่ง เมื่อ $n \to \infty$ ผลรวมจะเข้าสู่ค่าจริง

นิยามปริพันธ์ของรีมันน์
$$\int_a^b f(x)\,dx = \lim_{n \to \infty} \sum_{i=1}^{n} f(x_i^*)\,\Delta x, \quad \Delta x = \frac{b-a}{n}$$
ผลรวมรีมันน์: –  ·  ค่าจริง: –  ·  ความคลาดเคลื่อน: –
🔄 แอนติเดริเวทีฟ — ฟังก์ชันย้อนอนุพันธ์

ฟังก์ชัน $F$ เป็นแอนติเดริเวทีฟของ $f$ เมื่อ $F'(x) = f(x)$

ข้อสังเกต

ถ้า $F$ เป็นแอนติเดริเวทีฟของ $f$ แล้ว $F+C$ ก็เป็นเช่นกันสำหรับค่าคงที่ $C$ ใดๆ → ปริพันธ์ไม่จำกัด: $\int f(x)\,dx = F(x) + C$

เลื่อน $C$ เพื่อดูว่าครอบครัวเส้นโค้งทั้งหมดมีความชันเท่ากันทุกจุด (เพราะอนุพันธ์ของ $C$ คือ $0$)

🌉 ทฤษฎีบทหลักของแคลคูลัส (FTC)
FTC ภาค 1

ถ้า $A(x) = \displaystyle\int_a^x f(t)\,dt$ แล้ว $A'(x) = f(x)$ → การหาอนุพันธ์และการหาปริพันธ์เป็นการผกผันกัน

FTC ภาค 2

$\displaystyle\int_a^b f(x)\,dx = F(b) - F(a)$ เมื่อ $F$ เป็นแอนติเดริเวทีฟใดๆ ของ $f$

ลากแถบด้านล่างเพื่อดูว่าพื้นที่สะสม $A(x)$ สร้างตัวมันเองขึ้นมาอย่างไร

$f(t)$ กับพื้นที่จาก $a$ ถึง $x$

📘 สูตรปริพันธ์ที่ใช้บ่อย
$\displaystyle\int x^n\,dx = \dfrac{x^{n+1}}{n+1} + C, \ (n \ne -1)$
$\displaystyle\int \dfrac{1}{x}\,dx = \ln|x| + C$
$\displaystyle\int e^x\,dx = e^x + C$
$\displaystyle\int a^x\,dx = \dfrac{a^x}{\ln a} + C$
$\displaystyle\int \sin x\,dx = -\cos x + C$
$\displaystyle\int \cos x\,dx = \sin x + C$
$\displaystyle\int \sec^2 x\,dx = \tan x + C$
$\displaystyle\int k\cdot f(x)\,dx = k\int f(x)\,dx$
$\displaystyle\int [f(x) \pm g(x)]\,dx = \int f \pm \int g$
📝 สรุป — ปริพันธ์
  • $\int_a^b f(x)\,dx$ คือลิมิตของผลรวมรีมันน์เมื่อ $n \to \infty$
  • แอนติเดริเวทีฟ: $F' = f \Rightarrow \int f = F + C$
  • FTC: หาปริพันธ์จำกัดได้จาก $F(b) - F(a)$
  • จำสูตรหลัก: $x^n$, $1/x$, $e^x$, $\sin$, $\cos$, $\sec^2$
